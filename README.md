#  User Authentication System with Node.js

##  About

This project provides secure **Authentication Services** to users by offering **Signup/Register and Login/Logout APIs**.
It implements multiple layers of **authentication and authorization** to ensure maximum possible security.

The system is designed to follow real-world backend security practices, including:
- Secure password hashing
- Token-based authentication
- Protected routes
- Role-based access control
- Multi-factor authentication (MFA)
- Password reset mechanism

All APIs are built **from scratch**, focusing on security at every level of the application workflow.

---

## Technology Stack

This project is developed using the following technologies:

- **Backend:** Node.js, Express.js  
- **Database:** MongoDB Atlas (Cloud Database)  
- **Security:** bcrypt, JSON Web Tokens (JWT), dotenv, Speakeasy (MFA)  
- **Testing Tool:** Postman  

---

## Features

- User Signup / Registration API  
- Secure Password Hashing (bcrypt)  
- User Login with Token Generation  
- Token-Based Authentication for Protected Routes  
- Role-Based Authorization (User/Admin Support)  
- Multi-Factor Authentication (MFA)  
- Forgot Password & Reset Password Flow  
- MongoDB Atlas Cloud Integration  
- Secure Environment Variables using `.env`
